# File Reader Application

Create a File Reader Application for reading a file and printing its contents:
- allow users to specify the file to be read as a command-line argument
- implement error handling

Concepts covered:
- Introduction to Rust
- Variable assignment and immutability
- Basics of control flow
- Function basics
- Error handling
- File processing
